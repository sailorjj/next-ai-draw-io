# 图表自动布局 - Story

## 用户故事

作为一个使用图表功能的用户，我希望在 AI 生成图表后可以点击「整形」按钮，让图表自动重新排列布局，减少节点碰撞和重叠，使图表更整齐美观。

## 验收条件

- [ ] 图表生成完成后，聊天界面出现「整形」按钮
- [ ] 点击按钮后，当前图表重新布局
- [ ] 容器内子节点不再碰撞，保持 40px 间距
- [ ] 容器尺寸自动适配子节点 + 内边距
- [ ] 所有 cell 的 style、value、id 等属性不变
- [ ] 边连接正常，draw.io 自动路由
- [ ] 空图表时按钮点击无副作用
- [ ] 多次点击按钮结果一致（幂等）

## Ralph-Loop 执行模式

**迭代次数**：最多 3 次
**每次迭代**：实现 → 单元测试 → 修复失败 → 下一轮
**结束条件**：
1. 所有单元测试通过（无碰撞、间距正确、属性不变）
2. 代码编译无错误
3. 手动测试确认按钮出现在正确位置且点击有效
